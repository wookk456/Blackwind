'>' to increase pointer
'<' to decrease pointer
'+' to increase the value pointed by the pointer
'-' to decrease the value pointed by the pointer
'.' to print the value pointed by the pointer as the corresponding character
',' to change the value pointed by the pointer as user input
'[' if the value pointed by the pointer is 0 goto ']'
']' goto '['

example
input: ++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.
output: Hello World!
