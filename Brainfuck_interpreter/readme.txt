'>': increase pointer
'<': decrease pointer
'+': increase the value pointed by the pointer
'-': decrease the value pointed by the pointer
'.': print the value pointed by the pointer as the corresponding character
',': change the value pointed by the pointer as user input
'[': if the value pointed by the pointer is 0 goto ']'
']': goto '['

example
input: ++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.
output: Hello World!
